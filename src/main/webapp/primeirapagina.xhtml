Teste JSF com Jakarta EE

Sim, a dependência jakarta.jakartaee-web-api engloba as dependências legadas do JSF (JavaServer Faces) automaticamente quando você migra para o Jakarta EE. O Jakarta EE Web API inclui várias APIs Java EE, incluindo o JSF, Servlet, JSP e outras tecnologias web. Ao adicionar a dependência jakarta.jakartaee-web-api com escopo provided em seu projeto Maven, você está informando ao ambiente de execução (como um servidor de aplicativos compatível com Jakarta EE) que ele fornecerá essas APIs durante a execução, não sendo necessário incluí-las no artefato da aplicação (JAR/WAR/EAR).
